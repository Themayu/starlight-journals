# Starlight Journals

![Foundry v13](https://img.shields.io/badge/foundry-v13-green)

A Foundry VTT system module for *Starlight Journals* by Ian P. Quinlan.

## System Installation

This software is not yet packaged for installation into Foundry VTT by the
developers, and must be compiled from source.
